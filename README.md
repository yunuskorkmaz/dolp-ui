# dolp-ui

> 

[![NPM](https://img.shields.io/npm/v/dolp-ui.svg)](https://www.npmjs.com/package/dolp-ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dolp-ui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'dolp-ui'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
